# Test Case Document: Product Detail Page

| Field | Description |
| :--- | :--- |
| **Test Case ID** | TC_PDP_001 |
| **Test Scenario** | Verify the display of product information on the Product Detail Page (PDP) |
| **Test Case Description** | To ensure that all essential product details including name, price, stock status, images, and specifications are displayed correctly when a user navigates to a specific product page. |
| **Pre-Conditions** | 1. User is on the Home Page (https://ecommerce-playground.lambdatest.io/). <br> 2. Internet connection is stable. <br> 3. Browser is updated to the latest version. |
| **Test Steps** | 1. Navigate to the website: https://ecommerce-playground.lambdatest.io/ <br> 2. Scroll down to the "Featured" section or use the search bar to find a product (e.g., "HTC Touch HD"). <br> 3. Click on the product name or product image to open the Product Detail Page. <br> 4. Verify the Product Title. <br> 5. Verify the Price and Tax information. <br> 6. Check the Availability/Stock status. <br> 7. Verify the main product image and the gallery thumbnails. <br> 8. Check the "Description" and "Specification" tabs for content. |
| **Test Data** | Product Name: `HTC Touch HD` |
| **Expected Result** | 1. Product Title is correctly displayed as "HTC Touch HD". <br> 2. Price is clearly visible with the correct currency symbol ($). <br> 3. Availability status shows "In Stock" (or current actual status). <br> 4. Main image is clear and thumbnails change the main view when clicked. <br> 5. Description and Specification tabs contain accurate details of the product. <br> 6. Brand name (HTC) is displayed and clickable. |
