# Test Case Document: Product Detail Page - Product Comparison

| Field | Description |
| :--- | :--- |
| **Test Case ID** | TC_PDP_005 |
| **Test Scenario** | Verify "Compare this product" functionality |
| **Test Case Description** | To ensure that clicking the compare link adds the current product to the comparison list and triggers the appropriate notification. |
| **Pre-Conditions** | User is on the Product Detail Page of "HTC Touch HD". |
| **Test Steps** | 1. Locate the **"COMPARE THIS PRODUCT"** link (with the arrows icon). <br> 2. Click the link. <br> 3. Observe the top of the page for a success message. <br> 4. Click on the "Product Comparison" link within the success message or in the header/footer. |
| **Test Data** | Product: `HTC Touch HD` |
| **Expected Result** | 1. A success message appears (e.g., "Success: You have added HTC Touch HD to your product comparison!"). <br> 2. The comparison count in the header or sidebar updates. <br> 3. Navigating to the Comparison page shows the "HTC Touch HD" with its technical specifications side-by-side with other products. |
